## 背景

控制总览页的 L3/MC 曲线目前固定显示目标带、filtered、actual、raw 和控制事件中的若干项。用户在观察控制效果时，容易因为线条重叠、图例含义不清而难以识别目标值与实际控制值的关系。

## 目标

- 将控制总览中的目标带、filtered监控、actual实际、raw采样和控制事件改为用户可勾选显示层。
- 默认保持简洁：目标带、filtered、actual和控制事件开启，raw采样关闭。
- 用户切换显示层时不重新仿真，只改变当前前端渲染。

## 非目标

- 不改变仿真模型、监控采样、控制算法或导出数据。
- 不改变高级证据页的完整数据表。
- 本次不提交或上传 Git，等待用户后续统一上传。
