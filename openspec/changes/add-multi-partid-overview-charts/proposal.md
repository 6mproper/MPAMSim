## 背景

控制总览页的 L3/MC 主图目前只跟随单个当前 PARTID。用户在比较 noisy neighbor、BMAX/CMAX 控制效果或多 PARTID 目标带差异时，需要在同一张图里同时观察多个 PARTID 的目标、控制输入和实际结果。

## 目标

- 在控制总览新增独立的“同图对比 PARTID”多选控件。
- L3 与 MC 主图按该多选集叠加显示多个 PARTID。
- 图例明确区分“颜色=PARTID、线型/填充=曲线层”。
- 继续复用现有结果数据和图层开关，不新增仿真模型、数据通路或运行模式。

## 非目标

- 不修改 L3/MC 监控、控制或采样算法。
- 不改变 CPU/L3/MC 状态卡的单 PARTID 语义。
- 不新增阶段专用 UI 通路或导出格式。
